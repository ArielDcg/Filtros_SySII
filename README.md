# Filtros_SySII

Informe de diseño e implementación de filtros analógicos, IIR y FIR para el curso de **Sistemas y Señales II**.

## Estructura

- `docs/Informe_Filtros.tex` - Documento LaTeX del informe completo

## Compilación en Overleaf

Este repositorio está sincronizado con Overleaf. El archivo `Informe_Filtros.tex` se compilará automáticamente en Overleaf para generar el PDF.

Para acceder al proyecto:
1. Dirigirse al proyecto Overleaf vinculado
2. El compilador generará automáticamente el PDF
3. Consultar la pestaña "PDF" para descargar o visualizar
